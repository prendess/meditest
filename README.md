# 🏥 MediTest - Plataforma de Autoevaluación Médica

**MediTest** es una aplicación web responsiva diseñada para estudiantes de medicina y profesionales de la salud. Permite realizar tests de autoevaluación estructurados por especialidades y temas, con una interfaz limpia, moderna y adaptada a dispositivos móviles.

---

## 🚀 Características Principales

* **Multiespecialidad**: Organización de contenidos por ramas médicas como Dermatología y Pediatría.
* **Interfaz Sanitaria**: Diseño basado en colores verdes y blancos que evocan el entorno hospitalario.
* **Corrección Instantánea**: Sistema que valida las respuestas al finalizar el test, señalando aciertos en verde y errores en rojo.
* **Feedback de Calificación**: Cálculo automático de la nota sobre 10 al completar todas las preguntas.
* **Diseño Responsivo**: Experiencia optimizada para estudiar tanto en PC como en smartphone.
* **Aviso Legal**: Footer integrado indicando el carácter educativo de la plataforma.

---

## 🛠️ Tecnologías Utilizadas

* **React + Vite**: Para un entorno de desarrollo rápido y una aplicación de alto rendimiento.
* **Chakra UI**: Para una interfaz de usuario accesible y componentes modales elegantes.
* **React Router**: Gestión de navegación entre la página principal y los distintos exámenes.
* **React Icons**: Iconografía médica profesional (FaStethoscope, FaNotesMedical).
* **Git & GitHub Pages**: Control de versiones y despliegue automático.

---

## 📋 Contenidos Actuales

La aplicación cuenta actualmente con tests sobre:

### 🧴 Dermatología
1.  **Generalidades**: Capas de la piel, queratinocitos y melanocitos.
2.  **Exploración Física**: Signos clínicos como Nikolsky o Auspitz, y técnicas como la diascopia.
3.  **Lesiones Elementales**: Máculas, pápulas, vesículas y procesos como la liquenificación.

### 👶 Pediatría
1.  **Conceptos Básicos**: Fundamentos de la atención infantil.
2.  **Defectos Congénitos**: Patologías del desarrollo neonatal.
3.  **Dificultad Respiratoria**: Manejo inicial del recién nacido.

## 🔧 Instalación y Uso Local

Si deseas ejecutar **MediTest** en tu propia máquina:

1.  Clona el repositorio:
    ```bash
    git clone [https://github.com/prendess/meditest.git](https://github.com/prendess/meditest.git)
    ```
2.  Instala las dependencias:
    ```bash
    npm install
    ```
3.  Lanza el servidor de desarrollo:
    ```bash
    npm run dev
    ```
4.  Accede a: `http://localhost:5173/meditest/`
